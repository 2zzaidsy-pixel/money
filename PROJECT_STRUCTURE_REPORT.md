**Before (broken):**
```kotlin

    repositories { ... }

```

**After (fixed):**
```kotlin

    repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
```

## 4. Can the project run `./gradlew assembleDebug`?

### ❌ No — requires `google-services.json`

**Reason:** The `com.google.gms.google-services` plugin (line 5 of `app/build.gradle.kts`) requires `app/google-services.json` at build time. This file is generated from the **Firebase Console** and is **missing**.

### What is `google-services.json`?
- Contains Firebase project configuration (API keys, project ID, storage bucket, etc.)
- Generated at: Firebase Console > Project Settings > General > Your apps > Android > `google-services.json`
- Must be placed in: `PayWise/app/google-services.json`
- The app's `applicationId` in the file must match `com.paywise.app`
